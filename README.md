# Gail: The Mail Game

## 概要

オープンソースのブラウザゲームです。

使用言語: **Flatter**

## クイックスタート

```bash
npm install
npm run dev
```

1. **タイトル画面** - ゲームを開始
2. **チュートリアル** - 基本操作を学習
3. **ゲームプレイ** - 1ゲームループを体験
4. **スコア保存** - 結果を記録
