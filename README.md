# Gail

メールは魔物。敵を送信せよ。

## コンセプト
GailはFlatterで構築したオープンソースのブラウザゲームです。受信箱＝タワーディフェンスの世界観で、届くメールを捌きながら侵攻する敵を押し返します。

## クイックスタート
```bash
npm install
npm run dev
```
